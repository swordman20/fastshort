# Fast Short

开源的短剧 APP 解决方案，支持 IOS , Android , 微信小程序 , H5 , 抖音小程序。

> 计划未来支持Windows、MacOS、Linux等客户端。

* 支持类似抖音的随机播放
* 支持谷歌、Facebook、Apple、微信第三方登录
* 支持谷歌、PayPal、Apple、微信第三方支付
* 支持微信、微博、Twitter 分享
* 支持视频资源S3存储加速
* 支持自动下一集
* 支持用户留言交互
* 基本就是 ReelShort 有的功能咱都有
